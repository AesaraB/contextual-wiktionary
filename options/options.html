Wiktionary language

Blank means you are using the pre-set English Wiktionary. Only the pre-set has the customized display style.

Example values:

  • zh.wiktionary.org
  • fr.wiktionary.org
  • pl.wiktionary.org
  • en.wiktionary.org
    • This will result in the results displaying differently than the pre-set.

Styling isn't applied if using non-default. Instead, it will open the full Wiktionary page in the popup. Saves automatically.

PS. blame Mozilla if the text goes offscreen or similar . . .
